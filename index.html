                File format guide

                Upload a plain text (.txt) file with one study item per line.

                You can mix flashcards and multiple-choice questions in the same file.

                Flashcards / Matching
                Use this format for definitions and matching games:

                Term : Definition

                Example:
                Photosynthesis : Process plants use to make food

                Multiple Choice
                Use this format for quizzes:

                Question | Option A | Option B* | Option C | Option D

                Example:
                What is the capital of France? | Paris* | Rome | Berlin | Madrid

                Rules to remember:
                • Put only one card or question on each line
                • Add * at the end of the correct answer
                • Write “All of the above” exactly like that if you use it
                • Keep “All of the above” as the last option
                • Don’t split one question across multiple lines
